皇极经世是北宋哲学家邵雍创立的一套时间推演体系。他认为宇宙运行有固定的节律，就像四季交替一样，可以用数学模型来描述。

怎么看结果：
· 岁卦 = 今年的"主旋律"，类似星座的年度运势
· 能量值 = 阳气强弱，高=积极进取期，低=蛰伏蓄力期
· 五行关系 = 内外环境是否协调（生=顺，克=有阻力）
· 关键节点 = 未来可能出现转折的年份

层级从大到小：
元(12.96万年) → 会(1.08万年) → 运(360年) → 世(30年) → 岁(1年) → 月 → 日 → 时辰
就像"世纪→年代→年→月→日"，每一层都有对应的卦象描述当时的"气候"。
