## 数据录入原则

1. **查表不算表** — 老祖宗总结的规律直接存字典，不用公式推导
2. **按组合键索引** — 参考 bazi 项目 `sizi.py` 的模式，用"甲日甲子"这种组合键
3. **分层存储** — 基础数据(ganzhi) / 条文数据(texts) / 断语数据(duanyu) 分文件
4. **来源标注** — 每条数据标注出处典籍和卷次
